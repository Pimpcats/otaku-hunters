# sprites/pickups/

Floor pickup sprites (~16×16, transparent PNG, shown as-is). Filenames → slot:
- `heart_xp.png` — XP heart / fan-love (ハート)
- `word_tango.png` — word tile (単語)
- `gacha_capsule.png` — ガチャ evolution capsule
- `yen_coin.png`, `miren_spirit.png` — drop zones for future economy / SRS mechanics
  (not wired yet).

Add the path to `public/art-manifest.json` after dropping a file in.
