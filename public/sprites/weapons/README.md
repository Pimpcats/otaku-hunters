**Wired now (white on transparent → tinted in-engine):**
- `pocky.png` (Kōhai's お菓子 bolt, points +x, ~20×8)
- `shuriken.png` (Rōnin's 手裏剣, spun in-engine, ~16×16)
- `bullet.png` (generic bolt, ~10×10)

**Drop zone / not yet wired** (for future weapons — name them freely, e.g.
`pensaki_bolt.png`, `oen_wave.png`, `flash_burst.png`, `evo_*.png`): these have no
in-game weapon yet; wiring follows when the weapon is implemented. Add wired files
to `public/art-manifest.json`.
